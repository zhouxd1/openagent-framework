# Data Analysis Example

An AI agent specialized in data analysis and visualization.

## Features

- Query structured data
- Statistical analysis
- Data visualization
- Report generation

## Quick Start

```bash
npm install
export OPENAI_API_KEY=sk-xxxxxxxxxxxxx
npm start
```

## Usage

```
> What's the average age of users?
> Show me sales trends for last month
> Create a chart of user distribution by country
```
