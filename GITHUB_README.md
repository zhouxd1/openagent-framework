## English

OpenAgent Framework is a production-ready, enterprise-grade AI Agent framework for building, deploying, and managing intelligent AI applications.

### Key Features

- 🚀 **Multi-LLM Support** - OpenAI, Claude, DeepSeek, GLM, Ollama + 10 providers
- 🏢 **Enterprise Features** - RBAC, audit logs, multi-tenancy
- 🔄 **Agent Orchestration** - Multi-agent collaboration, workflow engine
- 📊 **Observability** - Distributed tracing, metrics, logging
- ☁️ **Production Ready** - Docker, Kubernetes, multi-cloud deployment
- ⚡ **Performance Optimized** - Worker Pool, connection pooling, lazy loading

### Quick Start

```bash
# Clone the repository
git clone https://github.com/zhouxd1/openagent-framework.git
cd openagent-framework

# Install dependencies
npm install

# Configure environment
cp .env.example .env

# Start with Docker
docker-compose up -d
```

## 中文

OpenAgent Framework 是一个功能完整、生产就绪的企业级 AI Agent 开发框架。

### 核心特性

- 🚀 **多LLM支持** - OpenAI、Claude、DeepSeek、GLM、Ollama 等 10+ 提供商
- 🏢 **企业级特性** - RBAC权限管理、审计日志、多租户支持
- 🔄 **Agent编排** - 多Agent协作、工作流引擎
- 📊 **可观测性** - 分布式追踪、指标监控、日志聚合
- ☁️ **生产就绪** - Docker、Kubernetes、多云平台部署
- ⚡ **性能优化** - Worker Pool、连接池、懒加载

### 快速开始

```bash
# 克隆仓库
git clone https://github.com/zhouxd1/openagent-framework.git
cd openagent-framework

# 安装依赖
npm install

# 配置环境
cp .env.example .env

# Docker 启动
docker-compose up -d
```
